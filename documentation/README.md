## 🛠️ Infrastructure Overview
- **Gateway (Port 5000)**: Entry point for all services.
- **Auth (Port 8000)**: Identity management.
- **Blog (Port 8001)**: Content engine.
- **Interactions (Port 8002)**: Social features.
- **Media (Port 8003)**: Asset management.
- **Real-time (Port 8004)**: WebSocket server.
- **Feed (Port 8005)**: Timeline service.
- **Search (Port 8006)**: Discovery engine.
- **Chat (Port 8007)**: Messaging service.
- **Worker (Port 8008)**: Background processing.
